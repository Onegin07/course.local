Структура файлов и папок раздела Чат:
	- images - папка с картинками
	- modules - папка модулей раздела
		- list-messages.php - вывод списка сообщений
		- list-user.php - вывод списка учеников
	- index.php - страница Чат
	- send-message.php - отправка сообщений 
	- remove_message.php - удаление сообщений
	- script.js - скрипты для обработки событий чата
