Структура файлов и папок общего раздела:
        - admin - папка админа(преподавателя)
        - chat - папка чат
        - configs - папка настроек и подключения к БД
        - images - папка с картинками
        - parts - папка с футером и хэдером главной страницы
	- becomeEventParticipant - файл отправки данных о выбраном мероприятии
        - edit_profile - файл редактирование профиля 
        - events - файл расписания
	- index - файл главная страница 
        - lessons-list - файл списка уроков
        - login - файл логин
        - logout - файл выхода
        - register - файл регистрации
	- script.js - скрипты для обработки событий
        - style -  файл стилей
